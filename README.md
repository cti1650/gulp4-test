# Gulp Test

## インストール方法

- Gulpのインストール  
  ```
  yarn add -D gulp
  ```

- Gulp プラグインのインストール  
  - gulp-sass プラグイン  
    ```
    yarn add -D sass gulp-sass
    ```

  - gulp-zip プラグイン  
    ```
    yarn add -D gulp-zip
    ```

## 実行方法

```
npx gulp
```
